Project Overview
Create a full-featured e-commerce application that supports Arabic (RTL), French, and English languages, running on both web (React) and mobile (React Native) platforms. Use the provided Figma design as the UI reference. Implement using MVC architecture with the code sharing strategy (Option 4).

Technical Requirements

2. MVC Architecture Implementation
Models (shared):

Product: id, name, price, images, description, category, stock

User: id, name, email, addresses, preferences

Cart: items, total, currency

Order: orderId, items, total, status, shippingAddress

Category: id, name, image, subcategories

Controllers (shared):

ProductController.js: Fetch products, filter, search

CartController.js: Add/remove items, calculate totals

UserController.js: Authentication, profile management

OrderController.js: Checkout, order history

LocalizationController.js: Language switching, RTL handling

Views (platform-specific):

Web: React components with CSS/Styled-components

Native: React Native components with StyleSheet
